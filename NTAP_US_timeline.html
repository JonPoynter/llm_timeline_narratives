2025-04-09 → 2025-12-02
In 2025, NetApp deepened direct integrations connecting its managed file services to leading AI platforms. With Google Cloud, NetApp Volumes advanced from planned Vertex AI connectivity to private preview and public confirmation that NetApp Volumes can serve as a governed data store for Gemini Enterprise, unlocking RAG and agentic patterns directly on enterprise data. Simultaneously, Azure NetApp Files introduced an Object REST API enabling native use of Azure AI services, including Azure OpenAI, without duplicating datasets. On AWS, Amazon S3 Access Points for FSx for NetApp ONTAP, plus Bedrock Knowledge Bases interoperability, made it easier to ground generative AI on existing file data. These steps collectively positioned NetApp as a multi-cloud data plane for AI.
