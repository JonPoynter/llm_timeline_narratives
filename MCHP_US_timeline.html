2026-02-05 → 2026-05-21
In early 2026 Microchip reported tangible traction in data center AI infrastructure components. Edge AI tooling and LLM-era interface foundation

2025-09-09 → 2025-11-06
Microchip moved to streamline embedded ML deployment and to expose its product data to LLM tools. In September 2025 it released tf2mplabh3, an open-source package to convert TensorFlow 2.x models into C for use in MPLAB X projects, lowering friction for on-device ML adoption across its MCU/MPU portfolio. In November 2025, Microchip announced a Model Context Protocol (MCP) Server that connects directly to LLM-compatible tools. This AI interface provides structured, up-to-date product data to copilots, chatbots, and enterprise agents—laying groundwork to surface Microchip content within leading LLM clients.

Early Google Cloud IoT integrations (foundational AI/ML enablement)

2018-10-10 → 2020-03-01
Microchip’s initial AI-adjacent moves centered on enabling embedded developers to connect devices to Google Cloud’s IoT and ML infrastructure. In 2018–2019 the company released AVR- and PIC-based IoT development boards that provided one-click connectivity to Google Cloud IoT Core. Supporting documentation and sandbox resources eased onboarding by shipping secure elements pre-registered to Microchip’s Google Cloud sandbox. In early 2020, Microchip published user guidance to integrate Google Home Assistant via Dialogflow/Google Actions and to route commands through Cloud Functions to IoT devices on Cloud IoT Core. These efforts established a pattern of product integrations with Google’s cloud AI ecosystem long before the current LLM wave.
