Foundational cloud security integrations (HSM) with major clouds

2019-03-04 → 2024-08-06

These integrations positioned Marvell hardware as a core building block for cloud key management and cryptographic operations underpinning enterprise workloads. By August 2024, Microsoft publicly confirmed that Azure updated its fleet of Marvell LiquidSecurity HSMs to FIPS 140-3 Level 3 for Azure Key Vault and Managed HSM, reinforcing Marvell’s role across hyperscaler security stacks. These relationships form an indirect but meaningful adjacency to AI/LLM services hosted on these clouds.
