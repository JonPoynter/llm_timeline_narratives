Foundational Google Cloud partnership and initial migration

2021-11-04 → 2023-12-31
CME Group and Google Cloud entered a 10-year strategic partnership on November 4, 2021 that coupled a $1 billion equity investment from Google with a plan to migrate CME’s technology infrastructure to Google Cloud. The agreement envisioned staged migration beginning with data and clearing, with Google’s analytics and machine learning capabilities positioned to support new real-time data, modeling, and risk tools. During 2023, CME advanced the program from planning to formal regulatory and client-facing milestones. In June 2023, CME filed an advance notice with the CFTC to migrate clearing and settlement systems to Google Cloud Platform, citing resiliency and security benefits from GCP’s global infrastructure. By July 2023, CME launched Reference Data API v3 hosted on Google Cloud and made historical referential data accessible via Google Cloud BigQuery, signaling the first visible productized components of the cloud transition for customers.
