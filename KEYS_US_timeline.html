Foundational LLM traffic analysis and API coverage

2023-02-21 → 2024-12-31
Keysight’s Application and Threat Intelligence (ATI) team began publishing technical analyses of large language model (LLM) application traffic, starting with OpenAI’s ChatGPT in early 2023 and expanding to Google’s Gemini in 2024. These efforts showcased Keysight’s capability to emulate and inspect LLM API behaviors in its traffic-generation and security-testing products, building credibility with networking and security customers validating AI workloads. By mid- to late-2024, Keysight had documented request patterns for OpenAI’s chat completions endpoint and introduced Gemini API superflows with configurable parameters such as model names and API versions. This laid the groundwork for broader multi-model test content and product integrations that followed.
